# node-xls-json

Converting xls file to json files using nodejs

## Install

```
  npm install node-xls-to-json
```

## Usage

```
  node_xj = require("node-xls-to-json");
  node_xj({
    input: "sample.xls", 
    output: "output.json"
  });
```
